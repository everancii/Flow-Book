## ADDED Requirements

### Requirement: Top 100 entry point on home screen
The system SHALL display a "4read Top 100" entry point on the home screen that allows users to navigate to the Top 100 screen.

#### Scenario: Entry point visible on home screen
- **WHEN** the user opens the home screen
- **THEN** a "4read Top 100" button or banner is visible in the 4read section

#### Scenario: Tap navigates to Top 100 screen
- **WHEN** the user taps the "4read Top 100" entry point
- **THEN** the app navigates to the `FourReadTopBooksScreen`
